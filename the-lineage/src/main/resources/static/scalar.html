How to authorize

  1. Call POST /auth/login with email + password (use the seeded dev users — see the README).
  2. Copy the accessToken from the response.
  3. Click the lock icon in the top-right of this page, choose bearerAuth, paste the token, then use Try it on any protected endpoint.

Token lifetime is short (15 min by default); refresh by calling /auth/login again. If Try it fails silently, your browser may be blocking requests to localhost:8080 — try the curl examples in the README instead.
